Project Architecture

ReviewPulse is a high-performance analytics dashboard designed to process and visualize large-scale e-commerce datasets. Built with a focus on real-time insights, it leverages a FastAPI backend and a MongoDB data lake to handle over 100,000 customer reviews with sub-second query latency.

The core engine utilizes a Logistic Regression model (trained on TF-IDF vectors) to achieve state-of-the-art sentiment classification. This allows businesses to instantly gauge customer satisfaction across thousands of interactions, transforming raw text into actionable intelligence.
